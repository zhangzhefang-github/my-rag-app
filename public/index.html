RAG 检索增强生成应用

欢迎使用

这是一个基于RAG (检索增强生成) 技术的演示应用。该应用能够根据问题检索相关文档，并基于这些文档生成准确的回答。

API 使用说明

本应用提供了两种查询接口：

1. 标准查询

POST /query

一次性返回完整的回答和引用来源。

2. 流式查询

POST /query/stream

使用服务器发送事件 (SSE) 流式返回生成的回答。

请求示例

{
    "query": "你的问题",
    "top_k": 3
}

技术栈

  • 后端: FastAPI
  • 语言模型: OpenAI API
  • 向量搜索: FAISS
